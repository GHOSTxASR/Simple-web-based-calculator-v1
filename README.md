# Simple Web-Based Calculator v1

A simple, responsive web-based calculator built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates basic frontend web development techniques to create a user-friendly, interactive calculator with essential mathematical operations.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division.
- **Responsive Design**: Works well on both desktop and mobile devices.
- **Simple User Interface**: Easy-to-use interface with buttons for each digit and operator.
- **Dynamic Font Size**: Automatically adjusts the text size if the calculator is resized to fit the container.

## Technologies Used

- **HTML**: Markup for structure.
- **CSS**: Styling and layout using Flexbox for responsiveness.
- **JavaScript**: Functionality and logic for calculations.

## How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Start performing calculations by clicking the buttons or using the keyboard (optional).

### Example Operations:
- `5 + 3 = 8`
- `9 * 2 = 18`
- `15 / 3 = 5`
- `20 - 4 = 16`

## Installation

If you'd like to run the project locally on your machine, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/simple-web-based-calculator-v1.git
    ```

2. Navigate to the project directory:
    ```bash
    cd simple-web-based-calculator-v1
    ```

3. Open `index.html` in your browser to view the project.
